# 🎓 Education Marketplace API

API backend construída com **NestJS** para uma plataforma de marketplace educacional.
Este projeto está em fase inicial e será expandido para oferecer recursos completos de gestão, autenticação e integração com serviços externos.

---

## 🚀 Stack Tecnológica

- **NestJS** (framework Node.js)
- **TypeScript**
- **ESLint + Prettier** (qualidade de código)
- **Jest** (testes)
- **Husky + lint-staged** (git hooks)
- **Commitlint** (commits convencionais)

---

## 📁 Estrutura do Projeto (inicial)

```
src/
├── main.ts
└── app.module.ts
```

> ⚠️ Esta estrutura irá crescer conforme módulos forem adicionados (ex.: usuários, cursos, pagamentos).
